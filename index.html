About This Project

This demonstration uses the Depth Anywhere model through a Gradio API. Frames from your webcam are sent to the API for processing and the resulting depth map is displayed.

The depth map shows estimated distances from the camera - brighter colors indicate objects closer to the camera.

Note: Processing happens on the server, so there may be some latency depending on your connection and server load.
